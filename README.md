## Lakers Homecourt
# by Sharkinov
Our website: https://hellfjrhbkbcs.my.canva.site/sharkinov

Project Lakers Homecourt seeks to create a web application for the LA Lakers. This platform will be focused on retaining fan engagementen during the National Basketball Association (NBA) off season, increase the team's audience, foster community around the sport, and create opportunity for monetization through sponsorship of fan events.
